Classic Windows UI themes for JetBrains IDEs.

Included Themes

  • Windows Classic – The Windows 95/NT/2000 look
  • Windows XP

Getting Started

  1. Go to Settings → Appearance & Behavior → Appearance
  2. Select Windows XP or Windows Classic from the Theme dropdown
  3. Click OK
